# 日本の言語学者と「世界的」言語学理論の関係

Last change: 2025/11/16-13:15:39.

## 概要

## 日本語研究の国際的射程と研究目的の多様性

日本語研究が国際的な一般言語学とどのように関係してきたかについては、筆者自身、長らく関心を抱いてきた。国際比較の観点から見ると、日本語研究が重視してきた対象や方法と、一般言語学が発展させてきた理論的枠組みのあいだには、目的の違いに由来するいくつかの非対称性が存在するように思われる。

日本語学の主要な伝統は、日本語という個別言語に固有の構造や運用の精緻な記述に重点を置いてきた。文末形式、助詞体系、間（ま）、呼応、あいづち、連鎖といった現象の綿密な観察は、日本語の特性を深く理解するうえで大きな成果をあげている。一方で、一般言語学が追究してきた数理的枠組みや普遍的制約のモデル化とは、研究目的や記述様式が必ずしも同一ではなかった。

このような方向性の違いは、研究の価値を減じるものではなく、むしろ言語学という学問領域が多様な視点を内包していることの表れである。ただ、結果として、日本語の現象をそのまま国際的な理論体系に翻訳するための接合点が十分に整理されてこなかった可能性はある。国際的な引用ネットワークに組み込まれる度合いにも、この研究目的の非対称性が影響してきたと考えられる。

この点を踏まえると、日本語研究と一般言語学の関係を再検討する際には、特定の研究伝統の評価ではなく、研究目的・記述形式・理論的前提の違いがどのように接続や翻訳を難しくしてきたのかという、より構造的な観点から議論することが有益であろう。

## 「世界的」であることの条件

現代の学術環境では、世界的に受容される理論は、以下の三要素を備えていることが事実上の要請となっている。

表: 「世界的」であることの三条件

| 条件             | 説明               |
| ---------------- | ------------------ |
| 数理的記述       | formalism          |
| 共有語彙         | shared terminology |
| 引用ネットワーク | citation network   |

この枠組みと比べると、日本語研究の多くは言語現象の精密観察や思想的分析を旨としており、形式化よりも現象の厚みを扱ってきた。そのため国際流通には不利であったが、言語の実態を捉える観点として独自の意義を持っていたことも確かである。

## 学際的研究としての即時文法・調整文法

即時文法と調整文法の枠組みは、こうした日本語研究の独自性を保持しつつ、認知科学・心理言語学における共有語彙を用いて発話の時間過程を記述しようとする試みである。発話の即時性や間合いに関わる現象を国際的に理解可能な形式へと接続することで、日本語を素材とする分析と普遍的な言語理論のあいだに橋を架けることを目指している。

## 言語普遍性の観点

即時文法の理論化において重要なのは、日本語で観察される現象を特定言語に限定された特殊性として扱うのではなく、言語一般に共有される生成過程の一側面として位置づけることである。日本語では呼応、あいづち、助詞終止、間（ま）などの即時的現象が顕著であるが、これらは発話生成の時間的構造を明瞭に映し出す一例にすぎない。

この際、同型性（isomorphism）の視点が有効である。形態・構文が異なる言語でも、驚き・確認・共感・否定などの短い応答が、文構造が未完のまま挿入されるという機能的な相同性が多くの言語で見られるからである。

## 即時文法の普遍的相同例

以下の例は、日本語に特有に見える現象が他言語でも機能的に対応していることを示す。

表: 即時的反応の言語間相同性例

| 機能         | 日本語の例             | 他言語の例（英語など）    |
| ------------ | ---------------------- | ------------------------- |
| 感情的応答   | えっ, うそ, まじで？   | Really? No way! What?!    |
| 呼応の端点   | で？, それで？         | So? Then?                 |
| 共感的反復   | そうそう, うんうん     | Yeah, yeah. Right, right. |
| 文未完の同意 | ..ね, ..さ, ..じゃん   | ..right? ..you know?      |
| 瞬間的反発   | いやいや, ちがうちがう | No, no. Wait, wait.       |

これらは語彙や構文形式こそ異なるものの、反応が即時的に生じ、未完の構文として挿入されるという時間構造において共通している。

## 理論的帰結

このような相同性が示されれば、日本語における即時的表現は、より普遍的な「即時文法」の具体的現れとして理解されうる。すなわち、immediate grammar in manifestation として日本語の現象を位置づけることで、即時文法は個別言語研究を超えて、言語全体に関わる生成原理を扱う理論として展開可能となる。

## 今後の研究課題

今後の実証的課題として、英語、中国語、フランス語、韓国語など複数言語における文未完応答や呼応、あいづちの構文的・語用的特徴を比較することが挙げられる。異なる言語類型において即時的反応がどのように実現されるかを整理することで、即時文法の時間構造がどの程度普遍的であるかを検証できる。

理論面では、即時文法の基盤となる発話生成過程をさらに多層的に整理する必要がある。"Universality of Immediate Grammar across Languages" を主題とした小節を設け、普遍的原理と個別言語の具体的現れの関係を体系的に論じることが望ましい。また、発話生成過程を「呼吸（行動的層）」「判断（認知的層）」「社会的共有（対人的・社会的層）」といった複数の次元から記述する拡張図表を用いることで、即時的反応が身体性・認知・社会性の交差するプロセスであることを示すことができる。
