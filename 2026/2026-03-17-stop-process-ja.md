# 処理文法モデルの最小公理系

Last change: 2026/03/18-14:43:01.

山元啓史, 東京科学大学

## はじめに

即時文法の再定式化を「処理文法モデル」として行うと、即時文法とは言語行動を可能にする低遅延の処理経路の体系になる。
この立場は、文法の誤りより、処理停止の方が本質的問題になる。
これは教育的にも重要であり、技能評価と直結する。
理論的な強みとして、処理文法を定義すると、認知科学と接続できる、SLA研究と対話できる、実験設計が可能になる、技能評価と直結するという利点がある。
これを「処理文法モデルの最小公理系」と呼ぶことにする。
たとえば、会話において、多少、冠詞が間違っているとか、三人称単数のsが抜けているとかよりも、止ってしまうことのほうが問題になる。
教育現場の実感としても、小さな形式誤りより、処理停止の方が重大というのは非常に妥当である。

## なぜ処理停止が本質的問題か

言語行動は時間依存の技能なので、流れが止まる、すなわち、機能が止まることを意味する。
一方、冠詞誤り、三単現の s 欠落は流れを止めない。
つまり、可用性に影響しない誤り、技能モデルで見ると表1のような誤りの影響になる。

表1：技能モデルにおける誤りの影響

| 現象     | 技能的影響 |
| -------- | ---------- |
| 冠詞誤り | 低い       |
| s 抜け   | 低い       |
| 語順揺れ | 中程度     |
| 処理停止 | 致命的     |

一方、技能とは、継続的処理能力であって、形式完全性ではない。
処理文法的に言うと即時文法は流れを維持するための文法であり、フィラー、言い換え、途中終了、不完全文、が許容される。国際的研究においても第二言語習得の研究領域では fluency、processing speed、breakdown rate が重要指標になっている。

すなわち、「誤りのない沈黙」より「誤りのある継続」の方が優れていると言える。
これは従来の文法教育観とかなり異なる見方を提供するものであろう。
理論的に言い換えるなら、文法能力とは正確性ではなく処理継続性であると言えよう。
