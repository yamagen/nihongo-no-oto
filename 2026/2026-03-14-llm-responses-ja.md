# LLMの応答にみられる二つの言語運用

Last change: 2026/03/15-15:37:33.

山元啓史, Ph.D. 東京科学大学

---

## 1. 問題の発端と応答スタイルの差の観察

LLMとの対話の中で現れた「二つの回答」、ここでは、ChatGPTが提示した、回答1と回答2という選択機能から端を発した議論について、観察された現象を整理する。
同じ質問に対してChatGPTが「短く共感的な回答」と「長く分析的な回答」という二つの異なるスタイルの応答を生成することが観察された。
LLMの応答の違いを整理すると、次のような対比が見えてくる。

表1: LLMの応答スタイルの特徴

| 応答の特徴 | 型     |
| ---------- | ------ |
| 短い説明   | 即時的 |
| 長い説明   | 分析的 |
| 共感的応答 | 会話的 |
| 分析的応答 | 構造的 |
| 会話的文章 | 連鎖的 |
| 構造的文章 | 階層的 |

ここでは、この違いが単なる文章の長さの違いではないことを指摘する。
認知科学における類似概念としては、直感的処理と熟考的処理の区別が挙げられる。この二つの処理様式は、LLMの応答スタイルの違いと非常によく対応している。Daniel Kahneman が示した System 1 と System 2 の区別も同様の概念を表している (Kahneman 2011)。このような二重過程モデルは、認知科学では dual-process theory として広く研究されている (Evans & Stanovich 2013)。
しかし、この理論は主に意思決定、判断の研究に用いられており、言語生成の構造として論じられることは少ない。

## 2. LLMはfast/slowを実装しているのか

LLMはfast systemとslow systemという二層構造を明示的に持っているわけではない。
ただし、LLM研究では、推論過程を段階的に生成させる chain-of-thought prompting など、生成スタイルの違いが報告されている (Wei et al. 2023)。
LLMは、大量の文章、確率的予測によって生成を行っている。
しかし、学習データには、会話、論文、説明文など様々な文体が含まれており、会話的生成、分析的生成という二つの言語運用のスタイルが自然に含まれる。
したがって、実務の世界における二層構造がLLMの生成に自然に反映されていると考えられる。
興味深いことに、この二層構造は多くの分野で暗黙に利用されている。
たとえば、ソフトウェアのユーザインタフェース、教育設計、人間工学などである。
多くの場合、直感的操作、分析的設定という二層で設計されている。
しかし、これは理論としてではなく、経験的な設計原理として扱われていることが多い。

## 3. 言語教育における偏りと時間幅

言語教育では、構造、文法規則が中心になる。
理由は単純で説明でき、テストできるからである。
言語生成の研究では、発話生成を複数の段階的過程として捉えるモデルが提案されており (Levelt 1989)、実際の言語運用には時間幅の異なる処理が存在することが示唆されている。
一方、即時的言語運用、会話の連鎖は体系化が難しいため、教育の中心になりにくい。
そこで、プロセス文法モデル(PGM)の導入を提案する(Yamamoto 2025)。
PGMでは、即時文法と調整文法という二つの言語運用を区別する。
この二つの文法は、時間幅と構造の違いによって特徴づけられる(表2)。

表2: PGMにおける二つの文法

| 文法     | 時間幅 | 構造 |
| -------- | ------ | ---- |
| 即時文法 | 短い   | 連鎖 |
| 調整文法 | 長い   | 構造 |

LLMの応答スタイルの差は、この二つの運用に非常によく対応している。
PGMにおいて重要なのは、二つの文法ではなく時間幅の違う言語運用である。
たとえば、会話は短い時間幅で流れ、論文は長い時間幅で構築される。
LLMの応答の違いもまた、この時間幅の差として理解できる。

---

## 4. 結論

LLMの応答の観察から次のことが見えてくる。
言語運用には、連鎖的処理、構造的処理という二つのモードが存在する。
この二つは対立するものではなく、時間幅の異なる言語運用プロセスとして理解できる。
この視点は、言語理論・言語教育・AI生成の三つを統一的に理解する可能性を持っている。
その意味で、人間の言語生成の二層化の実在性を示す重要な観察であると考えられる。
