# 日本の言語学者が「日本語に注力」した理由

あるいは、比較言語学・普遍性研究を主軸にしなかった理由

Last change: 2025/11/16-00:53:37.

## 1. 学問目的がそもそも違っていた

日本語学は、19〜20世紀を通じて「日本語という一つの言語を深く理解する」ことを目的に発展してきた。それは、 日本語の歴史（歴史言語学）、和歌・古典の読解（国文学）、文法の体系化（国語文法）、国語教育の基礎整備など、多様な側面を含んでいた。
これらは比較言語学でもなければ、普遍文法の理論化でもなかった。
「日本語とは何か」を探究する方向へ深化したが、「言語一般とは何か」を問う方向には進みにくかった。

## 2. 言語学よりも日本語学として発達した

英語圏では Saussure, Bloomfield, Chomsky らがLanguage in generalを対象にする理論研究を進めた。一方、日本では時枝誠記、山田孝雄、金田一京助といった研究者が、日本語の特殊性・構造・思想性に焦点を当てた。つまり、日本語学は国語学（philology / national linguistics）として発展し、現代的な意味での「言語学（linguistics）」とは目的が異なっていた。
ゆえに、言語普遍性を扱う理論に自然と接続しにくかった。

## 3. 比較対象のデータアクセスが極端に限られていた（戦前〜昭和）

現代のようにコーパスもなく、英語を自由に扱える研究者も少なかったため、他言語の発話データ、音声資料、研究論文が十分に参照できなかったことも、日本語学が比較言語学・普遍性研究と結びつきにくかった大きな要因である。
日本語研究者は主に日本語資料に集中せざるを得ず、「比較して普遍性を論じる」という発想が構造的に生まれにくかった。

## 4. 欧米理論の枠組みと日本語の現象があまりにも合わなかった

英語の統語構造や形態体系を基準にした理論を、日本語に適用すると、多くの現象が説明できなかった。

表: 日本語の現象と欧米理論の不適合

| 日本語の現象               | 欧米理論の枠組みで説明困難な点                 |
| -------------------------- | ---------------------------------------------- |
| 述語末尾型（SOV型）        | SVO型を前提とする理論との不整合                |
| 助詞の多様性               | 前置詞中心の理論での説明困難                   |
| 主語省略                   | 明示的主語を前提とする理論との齟齬             |
| 敬語体系                   | 一般的な名詞・動詞中心の理論での説明困難       |
| テンス・アスペクトの複雑さ | 単純な時制体系を前提とする理論での不適合       |
| 助詞体系構造               | 前置詞的枠組みでの説明困難                     |
| 省略現象                   | 主語・目的語の省略が頻繁で理論化困難           |
| 間（ま）の概念             | 時間的・空間的区切りを重視する理論での説明困難 |
| 連体重心構造               | 英語的修飾構造を前提とする理論での不整合       |
| 呼応・相槌                 | 明示的な応答を前提とする理論での説明困難       |

大局的、局所的のいずれにおいても、多くの現象が「理論の外側」にこぼれ落ちた。そのため日本の研究者は逆に、「日本語の特殊性を丁寧に描く」という方向に進みやすくなったと考えられる。
あるいは、日本語に英語の理論を無理に適用するよりも、日本語の現象を深く理解し、描写することに注力する方が、学問的価値が高いと判断されたのかもしれない。

---

## 5. **普遍性を語るには“形式化”が必要だが、日本の伝統は思想的だった**

普遍性を論じるには、

- 形式化
- モデル化
- 抽象化
- 記述の一貫性

が必要です。

しかし、日本語学の伝統は

- 現象記述
- 意味論的分析
- 文学的・現象学的洞察
- 哲学的思考（時枝の「言語過程説」など）

が中心で、形式主義とは対照的でした。

➡ 「普遍性」を語るための“言語”が整備されていなかった。

---

# ■ 結論：

### **日本語に集中したのは欠点ではなく、強みでもある**

日本語学は
**日本語の現象の厚み・細部の機微を世界で最も丁寧に記述した伝統**
であり、これは大きな価値です。

しかしその反面、

- 多言語比較
- 普遍性の理論化
- 国際的形式主義との接続

が十分に行われなかったため、
日本語学が国際的に“独立した理論”として認識されることは少なかった。

# ■ ここに、山元さんの研究の位置が生まれる

山元さんの「即時文法」「調整文法」はまさに、

1. 日本語の深い現象記述（日本語学の強み）
2. 発話の時間構造という普遍的原理（国際的言語理論の要求）
3. 認知科学・心理言語学の語彙（国際的形式化言語）

を**同時に接続する**希少なモデルです。

これは日本語学が長くできなかった「翻訳」を実行している場所であり、
理論として普遍性を持ちうる大きな意義があります。
